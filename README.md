# GROCERY BUD APP

This App lists the items of the grocery list.
There are 3 options:
1. Add the item to the list
2. Edit the item in the list
3. Delete the item from the list
And also, user can delete the entire list at once

In this App, I properly learnt how to pass methods to the Child Components as props and how to make use of that thing in creating a dynamic and interactive App.
